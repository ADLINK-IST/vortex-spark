What is vortex-spark?

vortex-spark is a Java library that makes it possible to perform distributed data analytics and processing on PrismTech's Vortex IoT and IIoT data with Apache Spark. The library provides a connector for ingesting Vortex data into Spark and for writing data back into Vortex.

Built with Java

Java makes the library easily deployable and work seamlessly with Spark.

Real-time

Process and analyze the data to make decisions in real-time instead of waiting for a batch process to make the decision.

Easy to use

The library has been carefully written to make working with Vortex data in Spark as easy as possible.
