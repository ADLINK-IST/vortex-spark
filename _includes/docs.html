Get Started

Create a Stream of Vortex Data

To consume data from Vortex we need to create a receive input stream.

JavaReceiverInputDStream<ShapeType> stream =
                VortexUtils.createStream(jssc, "Circle", "ShapeType", 
                                            ShapeType.class, 
                                            StorageLevel.MEMORY_ONLY());

Write Data to Vortex

To write data to Vortex we need to create a writer.

final int SHAPE_SIZE = 25;

sortedCounts.foreach(rdd -> {
    rdd.take(10).forEach(next -> {
        String color = next._2;
        try {
            getWriter()
                .write(new ShapeType(color, 
                                      next._1, 
                                      colorToY(color), 
                                      SHAPE_SIZE));
        } catch (TimeoutException e) {
            // handle exception
        }
    });
    return null;
});
